# Content Best Practices

Use this document to adapt the canonical K-AIDF best-practices guidance to editorial and publishing workflows.

## Focus Areas

- define audience, tone, and approved editorial boundaries
- separate drafting assistance from final human editorial approval
- require source verification for factual or high-stakes claims
- document review, approval, and update responsibilities
- keep traceability for prompts, edits, and publication decisions

## Operational Checks

- document content goals and success criteria
- keep reusable review checklists and style expectations explicit
- require sign-off before publishing externally visible changes
- review quality, consistency, and bias risks on each release cycle
